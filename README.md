  ## Installation
  
  No installation required. Visit the [deployed site](https://gentle-basin-90216.herokuapp.com/)
  
  ## Usage
  
  * Click 'login' and login
  * If you don't have a login, click 'Signup instead'
  * Click on a project from the homepage to view and add comments
  * Click on a project from your dashboard to edit the details
  * Browse or logout
